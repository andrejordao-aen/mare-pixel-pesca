Place student pixel art here.

If a PNG with one of these names exists, the game uses it automatically.
If it does not exist, the game keeps using the temporary Canvas drawing.

- cliffs.png
- sardinha.png

Tips:

- Use PNG files.
- Transparent background is best.
- Draw fish facing right. The game flips them when they swim left.
- Avoid spaces, accents and uppercase letters in file names.
